# AI Job Board - Frontend

React + Vite frontend for the AI Job Board application.

## Setup

```bash
npm install
npm run dev
```

Runs at http://localhost:5173. Ensure the backend is running at http://localhost:5001 (or update the Vite proxy in `vite.config.ts`).

## Scripts

- `npm run dev` - Development server
- `npm run build` - Production build
- `npm run preview` - Preview production build
